Useful tricks:

DP arrays

N, M, K = 2, 3, 5
dp1 = [0] * M 
dp2 = [[0] * M for _ in range(N)]    # to create N x M array.
dp3 = [[[0] * M for _ in range(N)] for _ in range(K)]   # K x N x M  array
